# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [3.0.0] - 2025-11-21

### Added

- End-user documentation for NotebookLM Extractor and Flashcard App
- VitePress for documentation
- Changelog, Privacy Policy and Terms of Service

## [2.0.0] - 2025-11-21

### Added

- GitHub Gist export functionality to NotebookLM extension

### Changed

- Restructure Extension into submodules
- Improve UI and save settings into local storage
- Enhance security with encrypting PAT

### Fixed

- API error handling
- File extension logic
- Authorization Token handling
- Error messages for end users

## [1.0.0] - 2025-11-21

### Added

- NotebookLM Extension
- Favicon for extension
- NotebookLM Output format

### Changed

- Refactor: Modularize and Convert Extension to TypeScript
- Update plugin styling

### Fixed

- Import issues

## [0.0.0] - 2025-11-15

- Imports
- Extract default markdown
